## О проекте

Групповой проект №5 «Deep Learning» курса «Современные методы анализа данных и машинного обучения», НИУ ВШЭ, 2025–26 учебный год.

Проект посвящён разработке единого DL-решения для страховой компании: предсказание риска страховых случаев по табличным данным о поведении водителя (полносвязные сети) и детекция повреждений кузова автомобиля по фотографиям (свёрточные и иные архитектуры глубинного обучения).
