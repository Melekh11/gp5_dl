# Данные

## О проекте

Групповой проект №5 «Deep Learning» курса «Современные методы анализа данных и машинного обучения», НИУ ВШЭ, 2025–26 учебный год.

Проект посвящён разработке единого DL-решения для страховой компании: предсказание риска страховых случаев по табличным данным о поведении водителя (полносвязные сети) и детекция повреждений кузова автомобиля по фотографиям (свёрточные и иные архитектуры глубинного обучения).

**Команда:** Масычев Александр, Мелехин Матвей, Яшина Ева, Хисамудинова Эмилия  
**Группа:** ББИ 231

---

В директории хранятся три набора данных, использованных в проекте.

## `car_clear_parsed` — собственный парсинг

Фотографии повреждённых автомобилей, собранные самостоятельно с помощью парсера ([drom_scraper.py](../drom_scraper.py)).
Изображения получены с открытых объявлений о продаже автомобилей и отфильтрованы вручную.

**Скачать:** [Google Drive](https://drive.google.com/drive/folders/155k-OqgV3Agn0aH1VIZa2HnO0jEYW2BW?usp=sharing) 

---

## `CarDD_release` — CarDD Dataset

Публичный датасет для задачи обнаружения повреждений кузова автомобиля.
Содержит аннотированные изображения в двух форматах:

- `CarDD_COCO` — аннотации в формате COCO (instance segmentation)
- `CarDD_SOD` — аннотации для задачи salient object detection

**Источник:** Xia, Z. et al. *CarDD: A New Dataset for Vision-Based Car Damage Detection*, 2023.
ResearchGate: https://www.researchgate.net/publication/369458280_CarDD_A_New_Dataset_for_Vision-Based_Car_Damage_Detection

**Скачать:** датасет распространяется авторами по запросу через страницу публикации на ResearchGate (кнопка *Request full-text* / контакт авторов).

---

## `driver_conditions` — Driver Conditions Dataset (Mendeley)

Датасет, описывающий условия вождения: включает `weekly_driving_profiles.csv` с агрегированными профилями поездок.
Используется для анализа паттернов эксплуатации автомобиля.

**Источник:** Mendeley Data, Dataset ID: `zf5zv7d8rs`, Version 1.
https://data.mendeley.com/datasets/zf5zv7d8rs/1

**Скачать:** открытый доступ — кнопка *Download All* на странице датасета.
